# Default Skills

These are the baseline skills every agent should receive before company-specific or role-specific skills are added.

The baseline layer is not just about reasoning or planning.

It also defines how an agent communicates clearly enough for humans to make decisions.

- `brainstorming`
- `canvas-communication` = visual explanation by default when structure is easier to see than read
- `connect-the-dots`
- `writing-plans`
- `verification-before-completion`
