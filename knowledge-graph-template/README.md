# Knowledge Graph Template

This is a starter skeleton for a curated knowledge graph.

It is not meant to mirror a full personal vault.

## Structure

- `_system/` = principles and curation rules
- `inbox/` = raw capture before promotion
- `curated/` = promoted durable notes
- `curated/examples/` = sample promoted notes

## Rule

Do not treat the knowledge graph as a dumping ground for every output.
