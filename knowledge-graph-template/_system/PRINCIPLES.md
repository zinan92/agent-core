# Knowledge Graph Principles

## 1. Curated, Not Exhaustive

Only promoted, durable knowledge belongs here.

## 2. Promotion Gate

Raw inputs should be reviewed before they enter the curated layer.

## 3. Durable Value

Good notes should stay useful across time, not just explain a momentary workflow run.

## 4. Compression Over Duplication

Prefer one strong canonical note over many weak duplicates.
