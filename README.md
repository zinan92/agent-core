# Wendy System Public Package

This package is a productized public export of the system core.

It is designed to help a new user understand and bootstrap an AI-native operating system without exposing the full internal operating environment.

## What This Package Includes

- core theory
- onboarding mechanism
- baseline default skills
- reusable templates
- a knowledge graph skeleton
- one demo agent
- example curated notes

## What This Package Does Not Include

- private conversations
- local runtime traces
- internal logs and cache
- the full internal knowledge graph
- the full internal employee roster

## Read Order

1. `PRINCIPLES.md`
2. `ARCHITECTURE.md`
3. `QUICKSTART.md`
4. `skills/agent-onboarding/`
5. `templates/`
6. `knowledge-graph-template/`
7. `examples/`

## Package Standard

This repository should make sense without requiring private context from the internal workspace.
