# Demo SOP

## Purpose

Convert incoming material into a compact, useful first-pass intelligence handoff.

## Inputs

- selected inputs
- source metadata
- handoff requirements

## Steps

1. read and filter incoming material
2. identify the highest-signal items
3. summarize into a downstream-ready packet

## Outputs

- concise handoff note
- clear next-action recommendation

## Escalation

- Trigger: unclear scope or low-confidence signal
- Escalate To: manager or owner
- SLA: same working cycle
